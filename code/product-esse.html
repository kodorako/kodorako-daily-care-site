Опис продукту

ESSE C5 Gel Cleanser — це м’який гелевий очищувальний засіб для щоденного догляду за шкірою. Формула ефективно видаляє забруднення, надлишки себуму та залишки макіяжу, не порушуючи природний баланс шкіри.

Завдяки делікатній дії гель очищає пори, освіжає та підтримує здоровий вигляд шкіри. Після використання шкіра залишається чистою, м’якою та без відчуття стягнутості.

Основні властивості

  • Делікатне, але ефективне очищення
  • Підтримка природного балансу шкіри
  • Освіжаючий ефект
  • Підходить для щоденного використання
  • Не пересушує шкіру
